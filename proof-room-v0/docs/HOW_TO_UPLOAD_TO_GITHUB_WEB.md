# How to upload the QUEBEC.AI Proof Room with GitHub web

1. Unzip the package.
2. Go to https://github.com/QuebecAI/QuebecAI.github.io.
3. Click **Add file → Upload files**.
4. Drag the `proof-room` folder into GitHub.
5. Use commit message: `Publish final QUEBEC.AI Proof Room`.
6. Click **Commit changes**.
7. Test: https://QuebecAI.github.io/proof-room/

Correct path: `proof-room/index.html`

Avoid: `proof-room/proof-room/index.html`
