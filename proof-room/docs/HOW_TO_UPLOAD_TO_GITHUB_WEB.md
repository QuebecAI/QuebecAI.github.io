# How to upload the QUEBEC.AI Proof Room using GitHub web interface

## Goal

Publish:

`https://QuebecAI.github.io/proof-room/`

## Best method

1. Unzip the final package.
2. Go to `QuebecAI / QuebecAI.github.io`.
3. Click **Add file → Upload files**.
4. Drag the folder named `proof-room` into GitHub.
5. Use commit message: `Publish final QUEBEC.AI Proof Room`.
6. Click **Commit changes**.

## Correct path

After upload, confirm GitHub shows:

`proof-room/index.html`

Do not create:

`proof-room/proof-room/index.html`
